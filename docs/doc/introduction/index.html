简介

湃锐（Cracknuts）项目旨在为用户提供一个“专业、易用、开源”的芯片安全分析解决方案。可对多种目标芯片进行功耗分析和电磁分析，并可实施时钟毛刺和电压毛刺故障注入攻击。项目的所有软硬件代码均将开源，方便用户二次开发和集成。

组成

项目由CrackNuts分析软件、Cracker测试设备和Nuts测试对象三部分组成。

  • CrackNuts使用Jupyter作为开发环境。
    • 支持设备和波形显示界面
    • 提供方便易用的API接口
    • 跨平台，Windows、Linux、Mac
  • Cracker包含一系列测试设备，可用于分析多种Nuts的安全性
    • Cracker-S1主要用于侧信道分析
    • Cracker-G1主要用于故障分析（开发中）
    • Cracker-F1主要用于硬件设计侧信道安全性验证（开发中）
    • 根据用户需求开发更多测试设备 ...
  • Nuts包含一系列目标芯片（可根据用户需求定制开发）
    • Nut-stm32f103
    • Nut-stm32f4
    • 根据用户需求开发更多待测设备 ...

部署方式

所有Cracker设备均支持POE方式工作，其优势在于：

  1. 极大简化部署方式，仅需一根网线就可实现设备供电和通信
  2. 突破物理距离限制，实现远程测试
  3. 方便组网，可同时控制多个测试设备
