关于我们

2024年7月1日 · 阅读需 5 分钟

安徽池芯新能源有限公司公司成立于2023年12月18日，由清华大学乌力吉教授创新创业团队创办，基于清华大学集成电路学院信息安全实验室积累，团队致力于打造关于硬件安全业内开源交流学习平台。

自创立之初便获得安徽政府数千万投资，公司专注于安全行业实验室配套设备与自主知识产权的车规芯片研发及销售，确保客户能以最优的性价比完成企业需求的搭建。

公司主要相关业务方向包括如下内容：

CrackNuts 硬件安全分析平台​

发源于清华大学集成电路学院乌力吉教授团队的信息安全开源平台。旨在为教师、学生、安全从业人员提供专业、易用、价格低廉的芯片安全分析工具，提供一个科研和技术交流平台，共同促进芯片安全技术发展。

CrackNuts硬件安全分析平台（开源平台）主要针对有硬件安全性检测需求的应用场景，旨在为企业研发/测试工程师、教师、学生、安全从业人员提供专业、易用、价格低廉的芯片安全分析工具，提供一个科研和技术交流平台，共同促进芯片安全技术发展。

开源理念：​

从根本上改变芯片安全的生态系统，让所有工程师都更容易获取或实现侧信道功率分析和故障攻击的理论和实践案例。任何人都可以免费的构建和使用它。

辅助硬件（Cracker S1）：​

为保护我们的客户能更好的使用我们的服务，需要您从官方申请或采购CrackNuts硬件设备，是经过彻底的内测设备，支持运行CrackNuts相关开源软件进行开发。

硬件主要指标（Cracker S1）：​

Cracker S1是一款针对物联网芯片的开源侧信道安全测试设备，支持多种通信接口，对外提供高质量功耗信号输出功能，同时，设备集成了高精度功耗采集电路和时钟控制电路，可实现测试对象（TOE）功耗信号的同步采样功能，设备提供Python API上位机控制接口。

  • TOE通信接口

    • UART
    • SPI
    • I2C
    • CAN
    • 时钟接口
    • 支持TOE更换和定制开发

  • TOE供电电压功能

    • 可调低噪音供电电路
    • 常用电压3.3V，调节范围2.0-4.5V，步进0.1V

  • 信号采集功能

    • 板载高信噪比功耗采集电路
    • 最大采样率62.5M Samples/S
    • 分辨率 12bit
    • 双通道，1路固定采集TOE功耗信号，1路可外接电磁探头采集电磁信号。
    • 双通道均支持动态增益调节功能，充分利用ADC分辨率
    • 支持TOE时钟、采样时钟关联调节，TOE时钟和采样时钟采用同源晶振，通过频率和相位调节实现高精度同步采样

  • 上位机接口

    • 千兆以太网
    • Python API
