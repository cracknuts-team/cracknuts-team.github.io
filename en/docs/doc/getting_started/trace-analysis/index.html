波形分析

在前面的数据采集章节，成功执行完运行模式后，采集到波形文件默认保存到当前笔记所在的文件夹下，默认为scarr格式，下面展示如何对曲线进行基本的产看以及读取。

新建笔记文件，并插入如下代码，执行后将显示波形展现的面板：

# 显示曲线分析面板
tap = cn.display_trace_analysis_panel()
tap

在新编辑框插入如下代码，打开曲线文件（曲线路径替换成您当前采集的波形数据文件路径），在波形面板将展示该文件的波形数据。

# 引入波形文件数据集对象
from cracknuts.solver.trace import ScarrTraceDataset
# 加载波形文件
scarr_trace_dataset = ScarrTraceDataset.load("dataset/20241205111924.zarr")
# 将数据集配置到波形显示面板
tap.set_trace_dataset(scarr_trace_dataset)
# 显示第二通道的前十条曲线
tap.show_trace[1, :10]

执行以下代码，可以获取曲线集合中的指定曲线，返回的是一个包含了曲线通道信息、索引信息、曲线本体、数据信息的元组，具体的格式请阅读API。

# 获取波形数据
scarr_trace_dataset.data[0,0] # 获取 channel 1 的第一条曲线
scarr_trace_dataset.data[1,:10] # 获取 channel 2 的前10条曲线

下面您就可以开始您的侧信道分析工作了。更多分析说明，请参阅教程。
